Student Grade Tracker

Overview

Student Grade Tracker is a Java-based console application designed to help users manage student academic records efficiently. The application allows users to enter student details, record marks, calculate grades, and display student performance in a simple and user-friendly manner.

Features

- Add student information
- Enter and manage student marks
- Calculate grades automatically
- Display student performance records
- Simple console-based interface
- Easy to understand and use

Technologies Used

- Java
- Object-Oriented Programming (OOP)
- VS Code

Project Structure

StudentGradeTracker.java

How to Run

1. Clone the repository:
   
   git clone https://github.com/Niharikakollati/CodeAlpha_StudentGradeTracker.git

2. Navigate to the project directory:
   
   cd CodeAlpha_StudentGradeTracker

3. Compile the Java file:
   
   javac StudentGradeTracker.java

4. Run the program:
   
   java StudentGradeTracker
